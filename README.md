# SKETCH TECH SOLUTIONS

Your trusted partner for comprehensive digital tech solutions.

## Quick Start (3 Steps)

**Terminal 1:**
```bash
cd client
npm install
npm run dev
```

**Terminal 2:**
```bash
cd server
npm install
npm start
```

Open: http://localhost:5173

## What You Get

✅ **Single Page** - All content on one page, no routing
✅ **8 Services** - Displayed with icons & descriptions  
✅ **Portfolio** - 4 featured projects
✅ **Contact Form** - With validation & success message
✅ **Features Section** - Why choose us with 4 benefits
✅ **Fully Responsive** - Mobile, tablet, desktop
✅ **Modern UI** - Gradient design, smooth animations

## Tech Stack

- **Frontend:** React 18, Vite, Axios
- **Backend:** Express.js, CORS, Node.js
- **Styling:** CSS3 (mobile-first responsive)

## Customization

**Change Colors:** Replace `#667eea` and `#764ba2` in App.css
**Update Services:** Edit services array in App.jsx
**Update Projects:** Edit projects array in App.jsx
**Update Features:** Edit feature items in App.jsx

## API Endpoints

- `POST /api/contact/submit` - Submit contact form
- `GET /api/services` - Get all services
- `GET /api/health` - Health check

## Deploy

- **Frontend:** Vercel
- **Backend:** Heroku, Railway, or Render
